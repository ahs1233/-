
# soug-frontend

واجهة المستخدم لتطبيق "السوگ." مبنية باستخدام Next.js + Tailwind CSS.

## 🚀 خطوات النشر على Vercel

1. ارفع هذا المجلد إلى GitHub
2. افتح [https://vercel.com](https://vercel.com) وسجّل الدخول
3. اربط المشروع بمستودع GitHub
4. أضف متغير بيئة باسم:

```
NEXT_PUBLIC_API_URL=https://your-backend-url.onrender.com
```

(استبدل الرابط برابط الـ Backend الخاص بك من Render)

5. اضغط Deploy!

📦 ستعمل الواجهة مباشرة مع البيانات من قاعدة البيانات.
